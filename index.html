Ramidus

Ramidus is a SPA for the modern web. Built with and for custom elements, Ramidus is a template that provides familiar conventions and a tiny bit of tooling to provide the best UX and DX possible.

This site is built with Ramidus and written in markdown.

Features

  • SPA routing using <app-root> and <a is="app-link"/> elements.
  • Layouts can be shared across multiple pages.
  • Write pages in HTML or GitHub-flavored markdown.
  • Familiar app structure, similar to Next or Nuxt
  • Vanilla folder structure and routing.
  • No building or tooling required.

Getting Started

npx ramidus@latest && npm run dev
